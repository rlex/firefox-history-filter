      Rule examples

      example.com matches only that hostname.

      @example.com matches that hostname and all subdomains.

      example.com:8080 matches only that port.

      192.168.1.1 matches an IPv4 address.

      [2001:db8::1] matches an IPv6 address.

      *.example.com matches one subdomain level.

      **.example.com matches any subdomain depth.

      https://example.com/private matches that URL prefix.

      *://*.example.com/*secret* supports wildcards.

      regex:^https://example\.com/(private|tmp)/ supports regex.
